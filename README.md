# Objection Handler / OpenAI Ads

A work sample by Tomi Gbeleyi for the OpenAI Client Partner, Ads Solutions role.

An interactive work sample: pick a vertical, get the objections advertisers actually raise about ChatGPT ads mapped to what is true, the counter, and the next step. Six sections per vertical, styled after my Stripe AE Prep Agent.

**View it:** open `index.html` in any browser. No build, no dependencies.

All data is illustrative. Named accounts are prospecting targets based on public information only; no OpenAI advertiser relationships are assumed or represented.
